# v4.4.4 Release Verification

Release date: 2026-07-21

GitHub release: https://github.com/TuLu-openclaw/tulu-openclaw-v2/releases/tag/v4.4.4

Release commit: `4649887f`

## Changes

- **Upgrade/Downgrade stuck bug fixed** — Sharp image dependency installation removed from the core upgrade command (restoring v4.4.1 behavior). Sharp is now best-effort: 120s timeout with failure only warning, never blocking the upgrade. Hardcoded npmjs.org registry references replaced with dynamic `get_configured_registry()` calls. New `install_openclaw_image_dependency_with_timeout` helper added.
- **Chat typing animation** — Decoupled network arrival speed from screen display speed with a typewriter buffer consumer. Streaming now renders pure text with line-break-to-br conversion; full markdown parsing is throttled to once every 180ms, eliminating O(n²) per-frame re-parsing. Status bar and watchdog timer updates throttled to 250ms, reducing localStorage writes and timer resets by ~100×.
- **Repository hygiene** — `.gitignore` updated with secrets, certificates, and debug script rules. `.gitattributes` added to track `_vendor/` binaries with Git LFS. `openclaw-version-policy.json` extended with 4.4.3 and 4.4.4 panel entries.

# v4.4.1 Release Verification

Release date: 2026-07-20

GitHub release: https://github.com/TuLu-openclaw/tulu-openclaw-v2/releases/tag/v4.4.1

Release commit: `2c88f661efbf409dc5bb3079f3507ef4f68f3086`

## Changes

- browser-use is pinned to `browser-use==0.13.6`, `mcp==1.26.0`, and `uv==0.7.12`.
- The runtime uses its product-isolated Chromium and does not silently fall back to system Edge.
- Health checks complete MCP `initialize`, `tools/list`, and `browser_get_state`, with a 45-second hard timeout and process-tree cleanup.
- Gateway connections support confirmed protocol versions v3 and v4 while preserving `connect.challenge` and Ed25519 device signatures.
- `PROTOCOL_MISMATCH` is treated as a terminal compatibility error instead of entering a reconnect loop.

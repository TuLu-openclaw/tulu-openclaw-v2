# OpenClaw 数字人模型目录

方案 C 使用这里的本地模型进行售卖版打包。

请放入：

- `openclaw-avatar.glb`（优先）
- 或后续扩展 `openclaw-avatar.vrm`

要求：

1. 必须拥有可商用授权。
2. 不要提交来源不明、禁止再分发的模型。
3. 模型文件名固定为 `openclaw-avatar.glb` 后，OpenClaw 主聊天页会自动加载。
4. 模型不存在时，会自动回退到 `public/digital-human/openclaw-avatar.svg`，不会影响应用启动。

当前仓库没有内置真人 GLB，是为了避免售卖版授权风险。
